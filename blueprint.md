
# Project Blueprint

## Overview

This project is a simple, framework-less web application for a lottery number generator. It allows users to generate a set of unique random numbers for a lottery draw.

## Implemented Features (v1)

*   **UI/UX:**
    *   A clean and modern user interface.
    *   A title "Lotto Number Generator".
    *   A container to display the generated lottery numbers.
    *   A button to trigger the number generation.
*   **Functionality:**
    *   Generates 6 unique random numbers between 1 and 45 upon button click.
    *   Displays the generated numbers in a visually appealing way.

## Current Task: Create Lotto Number Generator

*   **Steps:**
    1.  **Modify `index.html`:** Update the HTML structure to include a title, a container for the numbers, and a "Generate" button.
    2.  **Modify `style.css`:** Add styles for the overall layout, number display, and the button to create a visually appealing and modern design.
    3.  **Modify `main.js`:** Implement the JavaScript logic to generate and display 6 unique random numbers from 1 to 45 when the button is clicked.

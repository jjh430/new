# 오늘 뭐 먹지? (What shall we eat today?)

심플하고 깔끔한 디자인의 음식 메뉴 추천기입니다.

## 주요 기능
- **메뉴 추천**: 다양한 음식 메뉴 중 하나를 무작위로 추천해줍니다.
- **다크/라이트 모드**: 사용자의 취향에 맞는 테마를 선택할 수 있으며, 설정은 브라우저에 저장됩니다.
- **댓글 기능**: Disqus를 통해 사용자간의 소통이 가능합니다.

## 기술 스택
- HTML5
- CSS3 (Variables, Flexbox)
- JavaScript (Vanilla JS)

## 배포 방법 (Cloudflare Pages)
1. Cloudflare Pages에 저장소를 연결합니다.
2. 빌드 설정 없이 정적 파일로 배포합니다.
3. `https://choice-meal.pages.dev/`를 통해 접속할 수 있습니다.

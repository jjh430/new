# 로또 번호 추첨기 (Lotto Number Generator)

심플하고 깔끔한 디자인의 로또 번호 추첨기입니다.

## 주요 기능
- **번호 생성**: 1부터 45 사이의 중복되지 않는 6개 번호를 무작위로 추출하고 정렬하여 보여줍니다.
- **다크/라이트 모드**: 사용자의 취향에 맞는 테마를 선택할 수 있으며, 설정은 브라우저에 저장됩니다.
- **제휴 문의**: Formspree를 통한 간단한 문의 폼이 포함되어 있습니다.

## 기술 스택
- HTML5
- CSS3 (Variables, Flexbox)
- JavaScript (Vanilla JS)

## 배포 방법 (GitHub Pages)
1. GitHub 저장소의 `Settings` 탭으로 이동합니다.
2. 좌측 메뉴에서 `Pages`를 선택합니다.
3. `Build and deployment` 섹션의 `Branch`에서 `main` 브랜치를 선택하고 `Save`를 클릭합니다.
4. 잠시 후 상단에 생성된 URL을 통해 접속할 수 있습니다.
